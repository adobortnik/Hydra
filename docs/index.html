üêô What is Hydra?

Hydra is a mother/slave Instagram marketing system. It uses real Android phones with real Instagram accounts to grow your main (mother) accounts through automated interactions.

How It Works (Big Picture)

Slave accounts run on real phones

Each phone has multiple Instagram clone apps installed. Each clone = one logged-in account. These are your "slave" accounts.

Slaves interact with target audiences

The bots automatically follow people, like their posts, watch their reels, leave comments, share posts to story, and send DMs ‚Äî all to attract attention.

Attention flows to your mother account

When people check who followed/liked them, they see your slave account ‚Äî which promotes your mother account through bio links, story shares, and tags.

Mother account grows organically

The result: your main account gets real followers, real engagement, and real visibility ‚Äî all powered by your army of slave accounts.

üí° Why real phones?

Instagram detects emulators and fake devices. Hydra runs on real Samsung phones over WiFi, making every interaction look like a real person using a real phone. This is the safest approach for automation.

üöÄ Quick Start

Here's the fastest way to get up and running:

ü§ñ How the Bot Works

Understanding the bot cycle helps you configure it better.

The Cycle

For each device, the bot runs in a continuous loop:

Pick the next account

The bot rotates through all active accounts on the device. If an account is outside its time window, it's skipped.

Open the Instagram clone

The bot opens the specific clone app for that account (e.g., the "androif" clone for account #2).

Run enabled actions

Based on that account's settings, the bot performs actions: scroll feed, follow people, like posts, watch reels, share to story, etc. The order is randomized to look natural.

Cool down

After finishing, the bot waits 2 minutes before switching to the next account. Between individual actions, there are random delays (30-90 seconds).

Repeat

The cycle continues 24/7. Account 1 ‚Üí Account 2 ‚Üí Account 3 ‚Üí Account 1 ‚Üí ...

üí° Priority system

Not all tasks are equal. Scheduled posts always come first (they're time-sensitive). Then job orders (they're paid/assigned work). Then regular automation (follows, likes, etc.).

üö® Ban Prevention Tips

‚Ä¢ Warm up new accounts ‚Äî do only engagement (scroll, watch reels) for the first 3-5 days before enabling follow/like.
‚Ä¢ Don't max out every limit ‚Äî vary daily activity. Some days do 40 follows, some days do 20.
‚Ä¢ Use filters ‚Äî following empty/fake accounts doesn't help and may flag you.
‚Ä¢ Time windows matter ‚Äî don't run accounts 24/7. Set realistic hours (e.g., 8AM-11PM).
‚Ä¢ Watch the Health page ‚Äî if accounts start getting flagged, reduce limits immediately.

Filters

Filters help you follow quality accounts instead of bots and empty profiles:

  • Min/Max followers ‚Äî Skip accounts with 0 followers (likely bots) or millions (won't follow back)
  • Min posts ‚Äî Skip accounts with 0 posts (inactive or fake)
  • Has profile picture ‚Äî Skip accounts without a profile pic
  • Public only ‚Äî Skip private accounts (they can't see your content)
  • Bio keywords ‚Äî Whitelist/blacklist certain keywords in bios

üìÖ Scheduling

There are two types of scheduling in Hydra:

1. Content Scheduling

Schedule a post/reel/story to go live at a specific date and time. Set it up in Content Schedule, pick the account, attach the media, write the caption, pick the time. Done.

2. Account Time Windows

Each account has a start time and end time in Bot Settings. The bot only runs that account during this window. This simulates a real person who's only active during certain hours.

üí° Stagger your time windows

If you have 3 accounts on one phone, stagger them: Account 1 = 7AM-3PM, Account 2 = 10AM-6PM, Account 3 = 1PM-11PM. This way the phone is always doing something, and accounts get more natural-looking usage patterns.

Multi-Window Time Support

Accounts can have multiple active time windows per day using comma-separated values in Bot Settings. This lets you simulate different activity patterns ‚Äî like someone who checks Instagram in the morning and again in the evening.

SettingValueResult
start_time"2,12"Account runs 2:00-4:00 AM AND 12:00-2:00 PM
end_time"4,14"
start_time"8,18"Account runs 8:00 AM-12:00 PM AND 6:00-10:00 PM
end_time"12,22"
üí° Why multi-window?

Real people don't use Instagram for 16 hours straight. They check it a few times a day. Multi-window scheduling makes your accounts look more human by creating natural gaps in activity.

Work-Hour Aware Scheduling

The Content Schedule has a "Work-Hour Aware" strategy. When enabled, the scheduler reads each account's start_time/end_time and only schedules posts within their active window.

  • Prevents posts being scheduled at 3 AM when the account's window is 8 AM-11 PM
  • Shows work hour badges next to each account in the account selection, so you can see at a glance when each account is active
  • Automatically adjusts if you change an account's time window

Tag Dedup (Detailed)

Accounts sharing the same tag won't follow the same people. This prevents wasted actions when running multiple accounts targeting the same audience.

  • When Account A (tag: "fitness") follows @user123, that username is added to the "fitness" dedup list
  • When Account B (tag: "fitness") encounters @user123, it skips them automatically
  • The bot pre-loads usernames already followed by same-tag peers before each follow session
  • Settings are read fresh from the database each cycle ‚Äî no restart needed when you change tags
  • Different tags have separate dedup lists ‚Äî accounts in "fitness" and "travel" tags operate independently
‚ÑπÔ∏è Why use Tag Dedup?

If you have 10 accounts all following from the same source, without dedup they might all follow the same 50 people ‚Äî wasting 450 follow actions. With dedup, each account follows unique people, maximizing your reach.

‚ùì Frequently Asked Questions

How many accounts can one phone handle?
Up to 13 accounts per phone, each running in a separate Instagram clone app. In practice, 3-5 active accounts per phone gives the best performance.
Will my accounts get banned?
Risk exists with any automation. Hydra minimizes it by using real phones, human-like delays, randomized actions, and daily limits. Start slow with new accounts and follow the safety guidelines. Monitor the Account Health page.
Do I need to keep my computer on?
Yes ‚Äî the bot engine runs on your computer and controls the phones remotely via WiFi. If your computer shuts down, the bots stop. The phones just need to stay on and connected to WiFi.
What happens if a phone disconnects?
The bot automatically tries to reconnect. If the phone stays offline, that device's accounts pause until it's back. Other devices continue normally ‚Äî one offline phone doesn't affect the rest.
How do I add a new source account?
Go to Operations ‚Üí Manage Sources, select the account, type the source's Instagram username, and add it. The bot will start using it on the next cycle.
What's the difference between a source and a target?
A source is an Instagram account whose followers you want to reach. The targets are the individual followers the bot actually follows/likes ‚Äî they're automatically discovered from the source's follower list.
Can I see what the bot is doing right now?
Yes! Each device has a console window with colored live logs showing every action. You can also check Farm Stats for today's numbers, or watch the phones in real time via your device monitoring software.
How do I stop the bot?
Go to Automation ‚Üí Bot Manager and stop individual devices or the whole farm. You can also ask Jarvis to stop everything.
How do I change the dashboard password?
Go to the Settings page (/settings) and update the password in the Authentication section. You can also change it via the API endpoint POST /api/settings/password.
What is tag dedup and why should I use it?
Tag dedup prevents accounts with the same tag from following the same people. When Account A follows @user123, all other accounts with the same tag automatically skip that user. This eliminates wasted follow actions and maximizes your reach across the farm. Without it, 10 accounts might all follow the same 50 people.
Can I run the same account on different time windows?
Yes! Use multi-window support by setting comma-separated values. Example: start_time="8,18" and end_time="12,22" runs the account 8 AM-12 PM and 6 PM-10 PM. This simulates natural usage patterns ‚Äî checking Instagram a few times per day.
How does AI caption generation work?
In Content Schedule, type [AI] in the caption field to generate a unique caption for each post. Use [AI:your prompt here] for custom instructions. You need an OpenAI API key configured in Settings ‚Üí AI Configuration. Click "Preview AI" to see sample captions before scheduling.

üîß Troubleshooting

üî¥

Account keeps failing / 0 actions

Check if the Instagram clone app on that phone is working. Sometimes clone apps crash or the account gets logged out. Go to the phone and manually check the app. If it crashes instantly, the clone needs to be reinstalled.

üî¥

All sources marked as "dead"

If the clone app crashes, all source lookups fail, which falsely marks sources as dead. Fix the app issue first, then go to Manage Sources and reset the dead sources.

üî¥

Phone shows as disconnected

Check if the phone is on, connected to WiFi, and USB debugging is enabled. Try restarting the phone. Make sure it's on the same network as the computer (10.1.11.x).

üî¥

Dashboard won't load

The dashboard server might have stopped. Ask Jarvis to restart it, or launch it manually by double-clicking start_dashboard.vbs in the dashboard folder.

üî¥

Account getting rate-limited

Instagram is warning you to slow down. Reduce daily limits immediately (cut them in half). Let the account rest for 24-48 hours with only engagement (reels + feed scroll) before resuming growth actions.

üü°

Bot runs but doesn't follow anyone

Check: 1) Are follow sources set up? 2) Are the sources still active (not dead)? 3) Has the daily follow limit been reached? 4) Are filters too strict? Check Bot Settings and Manage Sources.
